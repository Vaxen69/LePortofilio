## Rappel rapide

- **Prod** : `index.html` (généré par `build-standalone.ps1`, ne pas éditer à la main)
- **Dev** : servir ce dossier en HTTP (`python -m http.server 8000`) et ouvrir
  [http://localhost:8000/Portfolio%20Roman%20Rodriguez.html](http://localhost:8000/Portfolio%20Roman%20Rodriguez.html)
- **Contenu** : `portfolio-data.jsx` · **Mise en page** : `variation-soft-cool.jsx`
- Après toute modification : `powershell -File build-standalone.ps1`
